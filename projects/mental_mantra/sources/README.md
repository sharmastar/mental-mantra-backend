# Mental Mantra

A mental wellness companion app.

## Getting Started

To run this app, you must complete the following configuration steps outside of code:

- **Google Sign-In on Android**: Register your SHA-1 fingerprint in the Firebase Console.
- **Google Sign-In on iOS**: Download `GoogleService-Info.plist` from the Firebase Console into the `ios/Runner/` directory.
- **Gemini AI**: Set the `GEMINI_API_KEY` in the `.env` file. You can obtain a free key from Google AI Studio (https://aistudio.google.com).

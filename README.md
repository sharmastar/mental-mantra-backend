# Mental Mantra 🧠✨

> An AI-powered, production-ready personalized mental wellness platform designed for holistic well-being, habit tracking, mood analytics, meditation, and crisis safety.

---

## 🌟 Features Overview

- **🤖 AI Mental Wellness Coach**: Personalized AI chat and journaling intelligence powered by Genkit & Gemini.
- **📊 Advanced Analytics & Scoring**: Real-time wellness index calculation using multi-vector scoring algorithms (mood, sleep, habits, hydration, screen time).
- **🛡️ Crisis Safety Engine**: Automated real-time keyword detection and immediate escalation protocols.
- **🧘 Meditation & Mindfulness**: Guided sessions, ambient sounds, customizable timers, and streak tracking.
- **🌙 Dark & Light Mode UI**: Premium modern glassmorphism design system built with custom animations and responsive layouts.
- **📱 Offline First**: Local caching and seamless data synchronization with Hive and secure storage.

---

## 🛠️ Technology Stack & Architecture

- **Framework**: Flutter 3.x / Dart 3.x
- **State Management**: Riverpod (`flutter_riverpod`)
- **Navigation**: GoRouter (`go_router`)
- **Backend Services**: Node.js, Express, Prisma, Firebase Authentication, Cloud Firestore
- **AI Engine**: Genkit / Google Gemini API
